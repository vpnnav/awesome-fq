# 贡献指南

感谢你愿意为 **Awesome FQ** 贡献内容！

## 提交规范

### ✅ 收录标准

- GitHub Star 数量 **≥ 1000**
- 与科学上网、代理协议相关（Clash / V2Ray / Xray / Shadowsocks / Trojan / Hysteria 等）
- 项目仍在维护，或有重要历史价值
- 开源协议明确

### ❌ 不收录

- 商业闭源软件
- 纯付费服务（机场）
- Star < 1000 的项目
- 与主题无关的项目

## 如何提交 PR

1. Fork 本仓库
2. 在对应分类下添加仓库信息（按 Star 数量从高到低排序）
3. 格式：`| [owner/repo](url) | 描述 | Stars | 语言 |`
4. 提交 Pull Request，说明添加原因
